== Polladillo

An easy-to-use app for live audience polling.  Easy poll setup, large, colorful results displays, live updates of your results as votes come in.  Your users can vote by text (great for lecture halls), or you can email a link to vote via the website.  

Built with Ruby on Rails, and Backbone with Twilio integration for text messages.
